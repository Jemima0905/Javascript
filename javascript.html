Function Declarations & Anonymous Functions

This syntax is called function declaration and anonymous function. It is hoisted by the browser when you initially start running your script. In other words, any named functions (aka function declarations) in your script are hoisted, or delivered, to the top of your script and interpreted before any other code is executed. In the end, you can call a function declaration from anywhere in your code.

 This syntax is called Object. basically an unordered collection of related data. It's a group of variables and functions that are thrown together to model or something you would likely recognize from the real world. In an object, variables and functions take on new names.

Accessing an Object (dot notation)

This syntax is called Accessing an object (dot notation) where in, you can access the properties or methods of an object using dot notation. You can also access properties using square brackets. To access a property or method within an object, you would reference the name of the object, followed by a period, then the name of the property or method.
